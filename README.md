# AutoGrade AI

AutoGrade AI is a full-stack web app that automatically evaluates student answer sheets using NLP.
It compares student answers with teacher answers and assigns marks based on semantic similarity, concept coverage, and relevance.

## Tech Stack

- **Frontend:** React (Vite)  
- **Backend:** Flask (Python)  
- **AI/NLP:** Sentence Transformers, CrossEncoder  
- **Tools:** PyPDF2  

## ▶️ Run Locally

```bash
#  Backend
cd backend
python app.py

#  Frontend
cd frontend
npm install
npm run dev
